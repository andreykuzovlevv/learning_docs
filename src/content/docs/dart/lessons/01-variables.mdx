## Задание

Программа "**Познакомимся!**"

> Эта программа помогает познакомиться с человеком.
Она задаёт вопросы и потом отвечает с персонализацией.

**Что делает программа:**
- Спрашивает: «Как тебя зовут?»
- Спрашивает: «Из какого ты города?»
- Спрашивает: «Сколько тебе лет?»
- Потом выводит приветствие используя свои знания о пользователе.



*Пример работы программы:*
``` text
Как тебя зовут?
Аня

Из какого ты города?
Москва

Сколько тебе лет?
10

Привет, Аня из города Москва! Тебе 10 лет.
```
